Unofficial ROBLOX Web API Documentation

What's Different?

This is basically the same thing as the one uploaded by GithubPagesStuff (original repo below) but with newer docs obtained from Ashburn.

Internal APIs (previously undocumented to the public), Backend APIs (previously completely undocumented, even to ROBLOX employees), and overall worse web design. Sorry.

How to Use?

  1. Get the browser extension 'CORS Unblock' (you won't have to do this for long)
  2. Press the button for the extension to use APIs here.
  3. Turn it off to go on normal ROBLOX.
